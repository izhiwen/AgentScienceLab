# Non-Goals

AgentScienceLab has strict boundaries. These are not temporary omissions; they define the scope of the project.

## Not A Framework

AgentScienceLab is not an agent framework. It does not define new primitives for agents, tools, memory, planning, task graphs, orchestration, or execution.

## Not A Benchmark Suite

AgentScienceLab is not a benchmark suite. It does not ship tasks, datasets, scoring scripts, harnesses, or benchmark protocols.

## Not A Leaderboard

AgentScienceLab is not a leaderboard. It does not rank models, agents, papers, systems, labs, or submissions.

## Not An Autonomous Scientist

AgentScienceLab is not an autonomous scientist. It does not promise end-to-end scientific discovery, automated publication, independent hypothesis generation, or unsupervised experimental execution.

## Not A Substrate Fork

AgentScienceLab is not a fork of AiPlus substrate behavior. It must not diverge from AiPlus routing, memory, locks, worktrees, lanes, MCP behavior, runtime launch, safety semantics, secret handling, dispatch semantics, status behavior, or refresh behavior.

## Not A Runtime System

AgentScienceLab is not a runtime system. It does not launch, supervise, schedule, sandbox, or execute agents outside the behavior already provided by AiPlus.

## Not A Replacement For Existing Agent Engineering Tools

AgentScienceLab is not a replacement for LangChain, AutoGen, CrewAI, AgentLab, Agent Laboratory, or other agent engineering tools. It can help researchers critique, document, and position work involving those systems, but it does not replace their engineering surfaces.
